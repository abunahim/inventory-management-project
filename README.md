# Inventory Management Project

## Project Goal

Built an inventory management system using Java Spring Boot while implementing DevOps practices.

## Tech Stack

- Backend: Java Spring Boot
- Database: MySQL
- Containerization: Docker
- CI/CD: GitHub Actions

## Features (Planned)

- Add products
- Update products
- Delete products
- Track stock quantity

## Project Phases

- Phase 1: Project Setup
- Phase 2: Spring Boot API
- Phase 3: Database Intergration
- Phase 4: Testing
- Phase 5: Docker
- Phase 6: CI/CD
- Phase 7: Deployment
